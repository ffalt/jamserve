# every-noise-genres.json

## Source

Genre coordinates are derived from [Every Noise at Once](https://everynoise.com) by Glenn McDonald
(formerly of Spotify's Music Intelligence team).

The visualization maps 5000+ Spotify-detected genres onto a 2D "sound space" where:

- **X-axis:** left = more organic / acoustic, right = more mechanical / electronic
- **Y-axis:** top = more atmospheric / diffuse, bottom = more energetic / spiky

Reference visualization and analysis:
[Ben Tanen - "Visualizing Musical Genres using Every Noise at Once"](https://observablehq.com/@ben-tanen/visualizing-musical-genres-using-every-noise-at-once)
